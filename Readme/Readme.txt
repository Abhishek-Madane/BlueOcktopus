# Project Name: BlueOcktopus

## Steps to Run Project

1. Open the solution `BlueOcktopus.sln` in Visual Studio.
2. Restore NuGet packages.
3. Update the database connection string in `appsettings.json` if needed.
4. Run the SQL script `DatabaseScript.sql` in SQL Server to create the database and tables.
5. Press F5 to run the project.

## Assumptions

- SQL Server is installed and running.
- Default database name is `StudentDB`.
- .NET Core 7.0 or higher is installed.
